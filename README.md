# aiproj01

Простий Python-проект для обчислення факторіалу чисел.

## Функціонал

| Компонент | Опис |
|-----------|------|
| `src/main.py` | Модуль з функцією `factorial(n: int) -> int`, яка обчислює факторіал невід'ємного числа `n` |
| `src/text1.txt` | Текстовий файл з українським текстом |

## Деталі
- **Вхідні дані:** ціле невід'ємне число `n`
- **Вихідні дані:** факторіал `n!` (добуток чисел від 1 до n)
- **Обробка помилок:** виключення `ValueError` для від'ємних чисел
